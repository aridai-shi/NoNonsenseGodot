Welcome to No-Nonsense Godot!

Plain and simple, it's a website for simple tutorials for the Godot Game Engine. I assume you've already downloaded it. If not, go download it, it's cool! This website isn't a good place to understand the Godot Engine. It's the place you go to get some code, some basic info and get things done! Got it? Good, grab yourself a tutorial from the sidebar on the left, remember to refer to the docs if you don't understand something, and start coding!
